# WEB-PORTOFOLIO

Website ini merupakan website portofolio pribadi yang menampilkan profil, keahlian, proyek, serta pengalaman organisasi dari Yoga Ari Anggoro sebagai seorang junior web developer dan graphic designer. Pada halaman utama ditampilkan perkenalan singkat mengenai diri, minat dalam bidang pengembangan web, serta foto profil. Website ini juga memiliki bagian Tentang Saya yang berisi informasi singkat mengenai latar belakang, pendidikan, dan keahlian seperti HTML, CSS, serta UI/UX. Selain itu, terdapat bagian Proyek dan Pengalaman Organisasi yang menampilkan beberapa proyek dan aktivitas organisasi yang pernah diikuti. Pada bagian akhir, tersedia halaman Hubungi Saya yang menyediakan formulir kontak agar pengunjung dapat mengirim pesan atau menjalin kerja sama. Dan dibagian akhir navigasi disediakan media sosial dari Yoga Ari Anggoro, agar memudahkan user menghubungi admin  jika sewaktu-waktu dibutuhkan. Website ini dibuat dengan tujuan untuk memperkenalkan profil, keterampilan, serta pengalaman yang dimiliki kepada pengunjung atau pihak yang tertarik untuk berkolaborasi.
